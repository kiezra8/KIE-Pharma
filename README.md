# SkieZ Pharma | Africa's Premier Medical Store

Welcome to the **SkieZ Pharma** web application. This project is a modern, high-end e-commerce platform built with React and Vite, inspired by the premium aesthetics of brands like Shein, tailored for the Ugandan medical market.

## 🚀 Quick Start

Due to local shell environment restrictions, please follow these manual steps to launch the app:

### 1. Project Initialization
Open your terminal in this directory and run:
```bash
npm install
npm run dev
```

### 2. Assets Setup (Important!)
I have generated professional high-end banners and product images for you. Please move them to the following locations in the project:

- **Banners** (Move to `public/banners/`):
  - `pharma_banner_1_1775152730666.png`
  - `pharma_banner_2_consumables_1775152772786.png`
  - `pharma_banner_3_lab_equipment_1775152837347.png`
  - `pharma_banner_4_branding_1775152855470.png`

- **Products** (Move to `public/products/`):
  - `product_gloves_premium_1775153047039.png`
  - `product_stethoscope_pro_1775153088958.png`

> [!TIP]
> You can find these generated files in your `.gemini\antigravity\brain\...` directory or in the artifacts panel.

### 3. Backend & Cloudinary
Update your keys in `src/firebase/config.js` to enable:
- **Firebase Firestore**: For product catalogs and accounts.
- **Firebase Storage**: For additional media.
- **Cloudinary**: For high-performance image optimization.

## ✨ Features
- **Shein-inspired UI**: Minimalist, high contrast, and smooth animations.
- **Hero Slider**: 4 dynamic animations showcasing key medical sectors.
- **Circular Categories**: 10 essential medical categories with a professional look.
- **Premium Product Grids**: "Essential & Trending" and "General Supplies" sections.
- **Integrated Chat**: Direct WhatsApp and Call links to specialists (+256702370441).
- **Mobile-First**: Fully responsive for the Ugandan market context.

## 🛠 Tech Stack
- **React 18** (Vite)
- **Framer Motion** (Stunning animations)
- **Lucide React** (Modern iconography)

- **Firebase & Cloudinary** (Robust backend/storage)
